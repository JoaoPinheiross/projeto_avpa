Este código projeto foi criado para a entrega de um trabalho semestral dos alunos do curso de Sistemas de Informação da Unip.
Nesse Github estão os códigos desenvolvidos para a APS, ao qual para acessar tem as seguintes maneiras.
Na pasta pycache: Estão os codigos em formato de bytecode, tendo que baixar os codigos para compilar na sua maquina.
Na pasta diagramas: Estão os diagramas, códigos python e imagens que serão geradas se baseando no uso no google colab. OBS: Antes de rodar o código, lembre-se de instalar o plantuml em uma das linhas do colab "pip install plantuml".
Os arquivos .py restantes são os projeto que roda a avaliação do projeto, sendo 4 modulos para o funcionamento, nessa ordem:
1 - instalar o plantuml "pip install plantuml".
2 - entradas.py
3 - Classes.py
4 - uml.py
5 - main.py
